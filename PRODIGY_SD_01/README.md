## Build a Temperature Conversion Program
This program converts temperatures between three major temperature scales: Celsius, Fahrenheit, and Kelvin. The user inputs a temperature value and its original unit of measurement, and the program converts it to the other two units.

## Features
Converts temperatures between Celsius, Fahrenheit, and Kelvin.<br>
Simple user interface for input and output.<br>
Easy-to-use format with prompts guiding the user through the process.<br><br>
