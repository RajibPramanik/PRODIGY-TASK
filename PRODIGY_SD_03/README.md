## Contact Management System
This program allows users to store, manage, and interact with contact information. It provides options to add, view, edit, and delete contacts. The contact information (including name, phone number, and email address) is stored in memory or a file for persistent storage, ensuring that the data remains even after the program is closed.

## Features
Add a New Contact: Users can input a contact's name, phone number, and email address to add to the contact list.<br>
View All Contacts: Displays a list of all stored contacts with their details.<br>
Edit Existing Contact: Users can update the contact information (name, phone number, or email address) of an existing contact.<br>
Delete Contact: Users can remove a contact from the contact list.<br>
Persistent Storage: Contact information can be stored in a file, ensuring that the data is saved even after the program is closed and reopened.
